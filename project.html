Team

Geometrical diagram is one of the categories under image accessibility classification as diagrams play a vital role in the understanding of geometry. In this project, we have opted for a template-based approach which will involve image and natural language processing. Image processing will focus on primitive shape detection along with its properties and associated labels. After detecting all the basic information, next step will focus on putting all this information together forming sentences while considering the context with the help of natural languages processing. Our main aim would be providing an alternative to the geometric diagram which will give a global picture which will contain the number of objects present and their positioning.

This project focuses on conversion of PDF to ePub which is initial processing required for obtaining desired standard format of digital content. Conversion of digital content involving text is easier but this approach focuses on extracting content in the correct HTML format. This will help to improve accessibility for tables, images, and mathematical content.

Experimental Setup diagrams is one of the categories under image accessibility classification as some of the concepts need setup visualization to understand. This approach is template based which requires image processing to detect objects and positionings which can help to understand the interaction among objects. This template will have natural language processing layer to generate the precise description of the diagram.
